# Pack Manager

Native `vim.pack` UI for this config.

Main features:

- Browse `vim.pack.get()` results in a two-pane float
- Inspect plugin metadata, lockfile revision, branches, and tags
- Jump from an installed plugin to the matching `lua/plugins/*.lua` config file
- Review a config snippet for the selected plugin
- Trigger `vim.pack.update()` in normal, offline, or lockfile-target modes
- Delete inactive plugins with confirmation
